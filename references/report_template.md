## 1. Рейтинг стимулов по сегментам

Для каждого сегмента исследования — стимулы, отсортированные по E[шкала] по
убыванию. Разделимость двух соседних стимулов считается по 95%-му бутстреп-ДИ
(`config.yaml: report.bootstrap_iters/seed/ci`): если интервалы не пересекаются —
разница устойчива; если пересекаются — разница в пределах шума метода, порядок
между ЭТИМИ двумя стимулами конкретно не считать надёжным (но это не отменяет
надёжность их совместного отрыва от третьего стимула, если такой есть).
