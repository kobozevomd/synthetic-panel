## 1. Рейтинг стимулов по сегментам

Для каждого сегмента исследования — стимулы в порядке убывания E[шкала]
(само значение — в приложении; здесь важен только порядок). Разделимость
считается через парный бутстреп на ОДИНАКОВЫХ профилях: P(A>B) — вероятность,
что стимул на месте выше действительно сильнее следующего по рангу, а не
просто оказался выше из-за шума метода. Три ярлыка (клиентский слой; полные
числа P(A>B) — в приложении):

- **уверенный разрыв** — P(A>B) ≥ 0,9: разница между этими двумя стимулами
  устойчива, можно опираться при выборе.
- **на грани** — 0,7 ≤ P(A>B) < 0,9: разница, скорее всего, есть, но не
  настолько уверенно, чтобы принимать решение только на ней — стоит смотреть
  на качественный разбор ниже или перепроверить на большей выборке.
- **в пределах шума** — P(A>B) < 0,7: порядок между ЭТИМИ двумя стимулами
  конкретно не считать надёжным (но это не отменяет надёжность их совместного
  отрыва от третьего стимула, если такой есть, — см. пример инверсии в
  `Инструкция_синтетическая_панель.md`, §6).

### Методология расчёта разделимости

Парный бутстреп разностей на ОДИНАКОВЫХ профилях (ресэмпл профилей, не
независимые доверительные интервалы двух стимулов по отдельности) —
`config.yaml: report.bootstrap_iters/seed/ci`. P(A>B) — доля бутстреп-сэмплов,
где стимул A действительно превзошёл B на тех же ресэмплированных профилях;
это устойчивее, чем проверка пересечения independent CI (см.
`references/methodology.md` за обоснованием). Пороги ярлыков секции 1 и этой
таблицы — 0,9 и 0,7 по P(A>B), см. раздел 1 выше.
